Description:  
An interactive HR Analytics Dashboard built in Power BI to visualize employee attrition trends, demographics, and job satisfaction insights for data‑driven HR decisions.

Project Description:  
This project is an interactive Power BI dashboard designed to analyze workforce data and highlight employee attrition patterns. It brings together key HR metrics such as Employee Count, Attrition Rate, Average Age, Average Salary, and Average Years of Service.
The dashboard includes visualizations for attrition by gender, education, age, salary, years at company, job satisfaction, and job role. With filters and interactive charts, HR teams can explore the data dynamically and uncover meaningful insights.

Purpose:  
The goal of this project is to help HR professionals quickly understand workforce trends, identify areas with high attrition, and make informed decisions to improve employee retention and satisfaction.

Tools Used:

Power BI

Excel/CSV datasets

Power Query for data cleaning

DAX for calculations

Outcome:  
A modern, interactive HR dashboard that highlights attrition drivers and supports better workforce planning and employee engagement strategies.
