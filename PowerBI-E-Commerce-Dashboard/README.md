**Project Description:**  
An interactive E‑Commerce Sales Dashboard built in Power BI to analyze sales performance, customer behavior, and profitability across categories and regions. 

It brings together key metrics like Total Sales, Profit, Quantity Sold, and Average Order Value (AOV) in a clean, modern interface, with visualizations for profit by state, monthly trends, category distribution, top customers, payment modes, and sub‑category performance.Filters for quarters and categories make it easy to explore the data dynamically and uncover meaningful insights.

**Purpose:**
The goal of this project is to make sales performance easy to understand at a glance. It helps identify which states and customers drive the most revenue, which product categories perform best, how profits vary month to month, and which payment methods are most popular.

**Tools Used:**

1.Power BI

2.Excel/CSV datasets

3.Power Query for data cleaning

4.DAX for calculations

**Outcome:**
A dark-themed, interactive dashboard that highlights important trends and supports better decision-making for e-commerce businesses.
