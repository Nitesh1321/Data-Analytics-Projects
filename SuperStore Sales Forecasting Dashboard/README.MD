Project Overview:  
An interactive Power BI dashboard developed for SuperStore sales forecasting, featuring dynamic visualizations of historical sales trends, 15‑day predictive analytics, and regional performance breakdown by ship mode. It consolidates key metrics such as Total Sales, Quantity Sold, Profit, and Average Delivery Time, while providing detailed breakdowns by segment, region, category, payment mode, ship mode, and sub‑category. With intuitive charts, filters, and real‑time insights, the dashboard empowers stakeholders to explore trends, compare performance, and make smarter, data‑driven decisions.


Purpose:  
The goal of this project is to help business stakeholders/
Track historical sales and profit trends
Forecast sales for the upcoming 15 days
Identify top‑performing regions, categories, and sub‑categories
Optimize shipping modes and delivery performance
Support strategic planning with real‑time insights

Tools Used:

Power BI

Excel/CSV datasets

Power Query for data cleaning

DAX for calculations and forecasting

Outcome:  
A modern, interactive dashboard that combines historical analysis with predictive forecasting, enabling SuperStore to make smarter, data‑driven decisions on sales, inventory, and regional strategies.
